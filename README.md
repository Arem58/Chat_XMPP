# Instructions:

To run the project just run the following command:

- run the main java file App.java
There, you can select an option at the main menu:
- register
- login
- exit

After you log in, the login menu will show up:
- 1 List of all users
- 2 Add users
- 3 Watch a user details
- 4 Communication with a user
- 5 Join a room
- 6 Change status
- 7 Sign out
- 8 Delete account

# Features:
- Register users
- Login
- List of all users
- Add users
- Watch a user details
- Communication with a user
- Join a room
- Send a message to a room
- Change status
- Notifications
- Sign out
- Delete account
